# SoftUni Exercises

A collection of labs, exercises, and exam preparations from my studies at [Software University (SoftUni)](https://softuni.bg/). Projects include **C#** (.NET 6 / .NET 8), **HTML & CSS**, **MS SQL**, and **ASP.NET Core**.

---

## Repository structure

### **Software-University**

| Course | Contents |
|--------|----------|
| **Fundamentals** | Basic syntax, data types, arrays, methods, lists, objects & classes, associative arrays, strings & text processing, regular expressions, mid-term and final exam prep |
| **OOP** | Inheritance, encapsulation, interfaces, polymorphism, exceptions & error handling, SOLID, reflection & attributes, unit testing, mocking & TDD, design patterns, exam preparation (6 exams) |
| **Entity Framework Core** | Introduction to EF, code-first, entity relations, migrations, LINQ, JSON/XML processing, advanced querying, MiniORM, ASP.NET essentials (e.g. MoviesApp), regular exam |
| **HTML & CSS** | Box model & typography, Flexbox layout, responsive web design (media queries), working with forms, exam preparation & regular exam |
| **MS SQL** | Database introduction, CRUD, table relations, built-in functions, subqueries & joins, indices & data aggregation, functions & stored procedures, exam |
| **ASP.NET** | ASP.NET Core MVC apps with Identity, EF Core, layered architecture (Web, Data, Services, ViewModels). Projects: **RecipeSharingPlatform**, **Horizons** (travel destinations) |

---

## Tech stack

- **Languages:** C#, HTML, CSS, T-SQL
- **Frameworks:** .NET 6, .NET 8, ASP.NET Core MVC, Entity Framework Core
- **Database:** MS SQL Server
- **Front-end:** HTML5, CSS3 (Flexbox, responsive design, forms)
- **Tools:** Visual Studio (`.sln`), SQL Server Management Studio / Azure Data Studio, xUnit for unit tests

---

## How to use

- **C#:** Open any `.sln` file in Visual Studio or run `dotnet run` from the project folder. Topics are organized in **Lab**, **Exercise**, and **More exercises** folders.
- **HTML & CSS:** Open any `.html` file in a browser or use a local server. Exercises cover layout, responsive design, and forms.
- **MS SQL:** Run `.sql` scripts in SQL Server Management Studio or Azure Data Studio against a SQL Server instance.
- **ASP.NET:** Open the solution (e.g. `RecipeSharingPlatform.sln`, `Horizons.sln`) in Visual Studio, set the Web project as startup, and run. Configure connection string in `appsettings.json` if using SQL Server.
